Introdução

Neste relatório você vai aprender a combinar texto e código em um único documento.

O objetivo é gerar um relatório com análise estatística usando Quarto no RStudio.

Pré-requisitos

  • R e RStudio instalados.
  • Quarto instalado (Menu Help → Check for Quarto Updates).
  • Para gerar PDF: instale o LaTeX leve (TinyTeX):
# Instalar TinyTeX (rode apenas uma vez)
# install.packages("tinytex")
# tinytex::install_tinytex()

Passo 1: Criando um projeto no RStudio

  1. Abra o RStudio.
  2. Vá em File → New Project → New Directory → New Project.
  3. Escolha uma pasta e dê um nome ao seu projeto, por exemplo: atividade-anova.
  4. Dentro do projeto, crie um novo arquivo Quarto (File → New File → Quarto Document) e salve como relatorio.qmd.

Passo 2: Inserindo código R

Use blocos de código para rodar análises estatísticas.
Um bloco de código em R é iniciado por três crases ``e a linguagem{r}`.

Exemplo:

Como executar códigos no Quarto

1. Rodar um bloco de código

  • Clique dentro do bloco e use o botão ▶️ (Run) no canto do bloco, ou
  • Atalho: Ctrl + Shift + Enter (Windows/Linux) ou Cmd + Shift + Enter (Mac).

2. Rodar linha por linha

  • Clique na linha desejada e use Ctrl + Enter (Windows/Linux) ou Cmd + Enter (Mac).

3. Renderizar o relatório inteiro

  • Clique em Render (botão azul no topo do RStudio). O Quarto executa todos os blocos em ordem e gera HTML/PDF.

💡 Dica: escreva o código neste .qmd. Assim seu relatório fica reproduzível e organizado.
